FOALFINDER LIVE AI UI

Denne version har:
- flottere premium UI
- live AI-generator
- lokal hestedatabase
- finder heste med samme far/morfar
- bruger matchende heste som inspiration til nye navne
- klar til senere Supabase-database

Sådan starter du:
1. Pak ZIP ud
2. Åbn mappen i VS Code
3. Omdøb .env.example til .env.local
4. Indsæt din OpenAI key
5. Kør:
   npm install
   npm run dev

Åbn:
http://localhost:3000

BILLEDER:
Læg dine egne hestebilleder i:
public/horses/

Brug fx:
horse-1.jpg
horse-2.jpg
horse-3.jpg
horse-4.jpg

Vigtigt:
Brug kun billeder du selv ejer eller har tilladelse til at bruge.
